# Rock Paper Scissors

A console-based Rock Paper Scissors game built with vanilla JavaScript as part of The Odin Project Foundations curriculum.

## How to Play

1. Open `index.html` in your browser
2. Open the browser console (`Cmd + Option + J` on Mac)
3. The game will prompt you to enter Rock, Paper, or Scissors for each round
4. After 5 rounds the final score and winner will be announced in the console

## Features

- Randomized computer choice each round
- Case-insensitive user input (rock, ROCK, Rock all work)
- Score tracking across all 5 rounds
- Win, lose, and tie detection for each round
- Final winner announcement after 5 rounds

## Technologies

- HTML
- JavaScript (Vanilla)

## What I Learned

- Writing and calling functions
- Using Math.random() for randomized logic
- Handling user input with prompt()
- Tracking state with variables across multiple function calls
- Nested functions and scope
